# NEXUS

A modern Discord bot built with **.NET 10**, designed for scalability, performance, and clean architecture.

## Features
* Built on **.NET 10**
* Powered by **NetCord**
* PostgreSQL support via **Npgsql**
* Clean and maintainable project structure
* Ready for future modular extensions

## Tech Stack
* **Framework:** .NET 10
* **Discord Library:** NetCord
* **Database:** PostgreSQL
* **Driver:** Npgsql

## Prerequisites
Before running the project, make sure you have the following installed:
* .NET 10 SDK
  Download: https://dotnet.microsoft.com/en-us/download/dotnet/10.0
* PostgreSQL *(optional, if database features are used)*
  Download: https://www.postgresql.org/download/

## Installation

Clone the repository:
```bash
git clone https://github.com/semi-constructor/NEXUS.git
cd NEXUS
```

Restore dependencies:
```bash
dotnet restore
```

Run the project:
```bash
dotnet run
```

## Dependencies

This project uses the following packages:

* NetCord
* Npgsql
